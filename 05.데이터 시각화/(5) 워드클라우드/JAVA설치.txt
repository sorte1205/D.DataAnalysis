JAVA 설치


	http://jdk.java.net/19
	권장버전은 19
	다운로드 파일을 C드라이브에 넣자. (압축 풀고, jdk-19.0.1을 옮겨야 한다)

윈도우 시스템에 경로 등록하기

	왼쪽/ 속성 -> 정보 -> 고급 시스템 설정 -> 환경변수 -> 시스템 변수 새로 만들기 -> 
	
		변수이름: JAVA_HOME
		변수값: C드라이브의 해당 폴더의 경로를 복사해 붙여 넣기 ex. C:\jdk-19.0.1
	
	시스템변수의 Patth 클릭 -> 새로만들기 -> C:\jdk-19.0.1\bin (bin파일경로) 붙여넣기
